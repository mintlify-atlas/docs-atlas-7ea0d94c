## Key features

This application demonstrates modern React development practices with production-ready features:

### Core functionality
- **CRUD operations** - Add, edit, delete, and toggle completion status for todos
- **Real-time search** - Debounced search by todo name with instant filtering
- **Priority system** - Assign priority levels from 1-10 to organize tasks
- **Subject categorization** - Group todos by subject (avtash, hagnash)
- **Date tracking** - Set and display due dates for each todo

### Technical highlights
- **Type safety** - Full TypeScript coverage with Zod runtime validation
- **Modern React** - Built with React 19 and functional components with hooks
- **Glassmorphic UI** - Beautiful Material-UI cards with backdrop blur effects
- **State management** - Uses Set data structure for efficient todo management
- **Form validation** - Client-side validation with Zod schemas
- **Debounced search** - Optimized search performance with 300ms debounce

## Type safety with Zod

The application uses Zod for runtime validation and TypeScript type inference:

```typescript
import z from "zod";

export const subjects = ["avtash", "hagnash"];

export const todoSchema = z.object({
  name: z.string(),
  subject: z.enum(subjects),
  priority: z.coerce.number().min(1).max(10),
  date: z.coerce.date(),
  completed: z.boolean().default(false),
});

export type Todo = z.infer<typeof todoSchema>;
```

<Tip>
  The `z.coerce` methods automatically convert form string inputs to the correct types (number and Date), making form handling seamless.
</Tip>
